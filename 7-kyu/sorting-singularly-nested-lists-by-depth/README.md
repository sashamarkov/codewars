Your task is to sort the main list in ascending order based on the depth of these subarrays.</p><p><strong>Input:</strong></p><ul><li>You will receive an array of arrays (a list of lists).</li><li>Each sub-array can only contain either another array or nothing (empty)</li><li>Main array might be completely empty</li></ul><p><strong>Understanding Depth</strong></p><p>The depth is zero-based:</p><ul><li>The elements in the main array have depth 1.</li><li>A single nested empty array (<code>[[]]</code>) has depth 2.</li><li>A double-nested empty array (<code>[[[]]]</code>) has depth 3, and so on.</li></ul><p><strong>Examples:</strong></p><p><code>[ [[[[[]]]]], [[]], [], [[[]]], []] </code>-&gt; <code>[ [], [], [[]], [[[]]], [[[[[]]]]] ]</code></p><ul><li><code>[]</code> has a depth of 1</li><li><code>[[]]</code> has a depth of 2</li><li><code>[[[]]]</code> has a depth of 3</li><li><code>[[[[[]]]]]</code> has a depth of 5</li></ul><p><code>[ [[]], [], [], [] , [[[[[[[]]]]]]], [], [[]] , [[[]]], [] ] </code>-&gt; <code>[ [], [], [], [], [], [[]], [[]], [[[]]], [[[[[[[]]]]]]] ]</code></p><p><code>[ [[]], [] ]</code>-&gt; <code>[[], [[]] ]</code></p><p>The final sorted order places the shallowest subarrays first and the deepest last.</p>
