<h2><a href=https://www.codewars.com/kata/69d3d8eac800a4e71c7e378d/train/python target="_blank">Recover My Code!</a></h2><h3>7 kyu</h3><h1 id="description">Description</h1><p>I lost my source code! All I have left are the tokens. Help me recover the source.</p><h1 id="tokens">Tokens</h1><p>Each token is represented as an array in the following format:</p><p><code>{command, arg1, arg2}</code></p><p>Example:</p><p><code>{"set", "x", "5"}</code></p><h1 id="commands">Commands</h1><p>Each valid command translates to a line of source code:</p><p><code>{"set", "x", "5"}</code> -&gt; <code>x = 5</code><br><code>{"add", "y", "3"}</code> -&gt; <code>y += 3</code><br><code>{"sub", "a", "4"}</code> -&gt; <code>a -= 4</code></p><h1 id="rules">Rules</h1><ul><li>The first argument is always a variable.</li><li>The second argument is either a number or a variable.</li></ul><h1 id="task">Task</h1><p>Given an array of tokens, return a string containing the reconstructed source code.</p><h1 id="constraints">Constraints</h1><ul><li>Tokens are given in the order they should appear in the source code.</li><li>Each command must be separated by a newline character (<code>\n</code>).</li><li>Invalid commands should be ignored.</li><li>Only <code>set</code>, <code>add</code>, and <code>sub</code> are valid commands.</li><li>Variables consist only of letters [a-zA-Z] and may be longer than one character.</li><li>Spaces in the output do not matter (<code>a=5</code>, <code>a =5</code> and <code>a = 5</code> are all valid).</li><li>There should not be any empty lines in the output.</li><li>If there are no valid commands, return an empty string.</li></ul>
