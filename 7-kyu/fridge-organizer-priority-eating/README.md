<h2><a href=https://www.codewars.com/kata/69971f385353edeaf428e1b0/train/python target="_blank">Fridge Organizer: Priority Eating</a></h2><h3>7 kyu</h3><h2 id="task">Task</h2><p>You have a messy fridge! To prevent food waste, you need to decide what to eat first based on two factors: <strong>expiry date</strong> and <strong>quantity</strong>.</p><p>Write a function that takes a list of food items and returns their names sorted by <strong>urgency score</strong>.</p><h3 id="data-format">Data Format</h3><p>Each food item is an <strong>object</strong> that has the following attributes:</p><ul><li><code>name</code>: The name of the food. (string)</li><li><code>expiry_days</code>: Days until it expires. (integer; can be negative)</li><li><code>is_almost_empty</code>: true if you only have a little bit left. (boolean)</li></ul><h3 id="sorting-rules">Sorting Rules</h3><ol><li><strong>Discard Spoiled</strong>: If the days to expiry are less than 0, the item is spoiled. <strong>Do not include it in the output.</strong></li><li><strong>Empty First</strong>: Items that are almost empty come before items that are not.</li><li><strong>Expiry</strong>: Within those groups, items with the <strong>lowest</strong> days to expiry come first.</li><li><strong>Alphabetical</strong>: If both status and expiry are the same, sort by name <strong>alphabetically</strong>.</li></ol><h3 id="example">Example</h3><p>When provided with the following list of food items:</p><table><thead><tr><th>Name</th><th>Days to Expiry</th><th>Is Almost Empty?</th></tr></thead><tbody><tr><td>Milk</td><td>3</td><td>false</td></tr><tr><td>Jam</td><td>3</td><td>true</td></tr><tr><td>Yogurt</td><td>1</td><td>false</td></tr><tr><td>Old Meat</td><td>-1</td><td>true</td></tr><tr><td>Today's Tofu</td><td>0</td><td>false</td></tr></tbody></table><p>You should return:</p><p><code>["Jam", "Today's Tofu", "Yogurt", "Milk"]</code></p><p>In that order.</p>
