<h2><a href=https://www.codewars.com/kata/69cda5b85599f307742ce70a/train/javascript target="_blank">How Many Lonely Letters?</a></h2><h3>7 kyu</h3><h2 id="task">Task</h2><p>Given a string, return the number of <strong>lonely</strong> letters it contains.</p><p>A letter is lonely when:</p><ol><li>It appears exactly once in the whole string.</li><li>Its alphabetical neighbors are both absent from the string.</li></ol><p>Alphabetical neighbors are the previous and next letters in the alphabet.</p><ul><li><code>d</code> has alphabetical neighbors <code>c</code> and <code>e</code>.</li><li><code>a</code> has only one possible neighbor: <code>b</code>.</li><li><code>z</code> has only one possible neighbor: <code>y</code>.</li><li>The alphabet is <strong>not cyclic</strong>, so <code>z</code> is not a neighbor of <code>a</code>, and <code>a</code> is not a neighbor of <code>z</code>.</li></ul><p>For example:</p><ul><li><code>d</code> is not lonely if <code>c</code> or <code>e</code> also appears somewhere in the text.</li><li><code>m</code> is lonely if <code>m</code> appears once and both <code>l</code> and <code>n</code> are absent.</li></ul><h2 id="rules">Rules</h2><ul><li>Ignore letter case.</li><li>Ignore all non-letter characters.</li><li>Work only with English letters <code>a-z</code>.</li></ul><h2 id="examples">Examples</h2><ul><li>Input: <code>"ad"</code> -&gt; Output: <code>2</code></li><li>Input: <code>"abc"</code> -&gt; Output: <code>0</code></li><li>Input: <code>"Hello, World!"</code> -&gt; Output: <code>3</code></li><li>Input: <code>"A-dA"</code> -&gt; Output: <code>1</code></li><li>Input: <code>"zz"</code> -&gt; Output: <code>0</code></li></ul>
