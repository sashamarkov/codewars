<h2><a href=https://www.codewars.com/kata/69b83710b26939b35fd10429/train/javascript target="_blank">Ship of Theseus</a></h2><h3>7 kyu</h3><p>The "Ship of Theseus" is a classic philosophical thought experiment about identity over time.</p><p>It asks: if every part of a ship is gradually replaced, one piece at a time, is it still the same ship in the end?</p><p>This kata turns that idea into a simple validation problem.</p><p>A ship is represented by a matrix of states.</p><p>Each row shows the ship at a different moment in time.</p><p>The ship is considered to remain the same only if, between every two consecutive rows, exactly one part of the ship has changed.</p><h2 id="rules">Rules</h2><ul><li>All rows must have the same length.</li><li>Rows must be compared <strong>position by position</strong>.</li><li>Between one row and the next, exactly one <strong>element (ship part)</strong> must be different.</li><li>If any transition changes <strong>zero elements</strong> or <strong>more than one element</strong>, the process is invalid.</li></ul><p>Return <strong>true</strong> if the whole process is valid, otherwise return <strong>false</strong>.</p><p>If the matrix has 0 or 1 row, return <strong>true</strong>.</p><hr><h2 id="example">Example</h2><p>Initial values:</p><pre><code class="language-text">[  ["a", "b", "c"],  ["x", "b", "c"],  ["x", "y", "c"],  ["x", "y", "z"]]</code></pre><h3 id="step-by-step">Step-by-step</h3><pre><code class="language-text">["a", "b", "c"] -&gt; ["x", "b", "c"] -&gt; 1 change["x", "b", "c"] -&gt; ["x", "y", "c"] -&gt; 1 change["x", "y", "c"] -&gt; ["x", "y", "z"] -&gt; 1 change</code></pre><h3 id="result">Result</h3><pre><code class="language-text">true</code></pre><hr><h2 id="another-example">Another example</h2><p>Initial values:</p><pre><code class="language-text">[  ["a", "b", "c"],  ["x", "y", "c"]]</code></pre><h3 id="step-by-step-1">Step-by-step</h3><pre><code class="language-text">["a", "b", "c"] -&gt; ["x", "y", "c"] -&gt; 2 changes</code></pre><h3 id="result-1">Result</h3><pre><code class="language-text">false</code></pre><hr><h2 id="different-row-lengths-example">Different row lengths example</h2><p>Initial values:</p><pre><code class="language-text">[  ["a", "b", "c"],  ["x", "b"]]</code></pre><h3 id="step-by-step-2">Step-by-step</h3><pre><code class="language-text">The rows have different lengths, so the process is invalid.</code></pre><h3 id="result-2">Result</h3><pre><code class="language-text">false</code></pre>
