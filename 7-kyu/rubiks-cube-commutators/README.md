<h2><a href=https://www.codewars.com/kata/6939b1596fb89b7ec3d128ba/train/python target="_blank">Rubik's Cube Commutators</a></h2><h3>7 kyu</h3><p>A <strong>commutator</strong> is a type of algorithm that takes this form:</p><ul><li>A</li><li>B</li><li>A backwards</li><li>B backwards</li></ul><p>These types of algorithms are very useful when solving the Rubik's Cube, so I need you to code a function that creates them.</p><p>Rubik's cube notation is thus:</p><ul><li>Each side of the cube has a corresponding letter: R, L, U, D, F, B.</li><li>Each letter has 3 variations:<ul><li>R: Turn the side 90 degrees clockwise.</li><li>R': Turn the side 90 degrees counterclockwise.</li><li>R2: Turn the side 180 degrees. (Note that R2' is not a valid turn.)</li></ul></li></ul><p>Your function must take in 2 strings (A and B) with turns separated by spaces, and it must output a string of the same format with the completed commutator <strong>in full.</strong> (<em>e.g., do not shorten the output as in <a href="https://www.codewars.com/kata/68b6aecf7dae381e330a1021" data-turbolinks="false" target="_blank">Aza_rt's kata</a></em>)</p><p>Example:</p><ul><li>A: "R U"</li><li>B: "F' L2"</li><li>Result: "R U F' L2 U' R' L2 F"</li></ul><p><em>Note: When reversing a sequence, you must reverse the <strong>direction</strong> of the turns as well as the <strong>order.</strong></em></p>
