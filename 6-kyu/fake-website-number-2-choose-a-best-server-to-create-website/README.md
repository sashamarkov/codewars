This time, he needs to choose a best server to create his website.</p><p>What is a best server? John mainly from two aspects to consider:</p><pre><code>One is the performance, including network delay and disconnectionrate; The other is the price. that is, the annual rent of the server.</code></pre><p>John has selected some servers, asked the annual rental price of these servers, and each server for 1 day of performance testing (24 hours).  Get one test data per hour, test data is the value of network delay. Unit: millisecond.</p><p>Please help John write the function <code>bestServer</code> to find the best choice.
