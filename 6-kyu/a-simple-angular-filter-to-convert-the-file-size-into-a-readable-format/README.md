<h2><a href=https://www.codewars.com/kata/56d56177e59d0e0a5a000677/train/javascript target="_blank">A simple Angular filter to convert the file size into a readable format</a></h2><h3>6 kyu</h3><p>This Kata gives you the possibility to create a very simple, but useful filter for Angular. We are going to implement the filter inside the angular app controller.</p><p>The Angular app renders different files and shows them inside a simple table. The last change timestamp is parsed over the angular date filter. So here we use the Angular standart.</p><p>The size of each file is given in bytes. That value is not very easy to read. The output 97.69 MB is much nicer then 102438491. Angular offers therefore the possibility to create a custom filter. Inside the html we are going to write:</p><pre><code class="language-html">&lt;tr ng-repeat="item in myFiles"&gt;  &lt;td&gt;{{item.filename}}&lt;/td&gt;    &lt;td&gt;{{item.lastchange | date : 'dd.MM.yyyy HH:mm:ss'}}&lt;/td&gt;    &lt;td&gt;{{item.size | fileSize}}&lt;/td&gt;  &lt;/tr&gt;</code></pre><p>You can find a litle bit more information about the filters at the angular docs:</p><ul><li><a href="https://docs.angularjs.org/tutorial/step_09" data-turbolinks="false" target="_blank">Angular Tutorial</a></li><li><a
