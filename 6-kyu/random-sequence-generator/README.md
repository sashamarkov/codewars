<h2><a href=https://www.codewars.com/kata/5302b069ad5217ce2d0000f1/train/javascript target="_blank">Random Sequence Generator</a></h2><h3>6 kyu</h3><p>In this kata, you have to complete the code to generate a sequence of random <strong>not repeated</strong> numbers between the minimum and maximum.</p><p>Complete:</p><ul><li>The <code>Randomizer(min, max)</code> constructor creates a shuffled array of <code>[min..max]</code></li><li>The <code>next()</code> method: returns the next random number</li><li>The <code>sequence(size)</code> method: returns an array of length <code>size</code> of random numbers. If no <code>size</code> is passed, returns the remaining random numbers.</li></ul><p>For example:</p><pre><code class="language-javascript">  <span class="cm-keyword">var</span> <span class="cm-def">r</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">Randomizer</span>(<span class="cm-number">1</span>,<span class="cm-number">6</span>);  <span class="cm-variable">r</span>.<span class="cm-property">next</span>(); <span class="cm-comment">// 3 (or any number between 1 and 6)</span>  <span class="cm-variable">r</span>.<span class="cm-property">sequence</span>(<span class="cm-number">2</span>); <span class="cm-comment">// [1, 6] (two number between 1 and 6, except 3)</span>  <span class="cm-variable">r</span>.<span class="cm-property">sequence</span>(); <span class="cm-comment">// [2, 4, 5] (the remaining numbers)</span></code></pre><p>It is assumed that <code>min</code> and <code>max</code> are always numbers and <code>min &lt;= max</code>. You do not  have to worry about checking this.</p><p>An exception will be thrown if:</p><ul><li>Call <code>next()</code> and no more numbers remain</li><li>Call <code>sequence(size)</code> and size is greater than the number of numbers remaining</li><li>Call <code>sequence(size)</code> and no more numbers remain (regardless of size being passed or not)</li></ul><p>The specific error message is not important.</p>
