<h2><a href=https://www.codewars.com/kata/65f1c009e44a0f0777c9fa06/train/python target="_blank">Collapse Left</a></h2><h3>6 kyu</h3><h1 id="task">Task</h1><p>Write a function that takes an array containing numbers and functions. The output of the function should be an array of only numbers. So how are we to remove the functions from the array?</p><p>All functions must be applied to the number before it prior to the function being discarded from the array. It is as if the functions are collapsing left in the array modifying the first number to the left of the function. The functions should apply in order of index in the array. After a function has been applied to the number before it, it should be "removed" from the array to allow any following functions to also modify that new number.</p><h1 id="example">Example</h1><pre><code class="language-javascript">[ <span class="cm-number">3</span>, <span class="cm-variable">f</span>(<span class="cm-variable">x</span>) <span class="cm-operator">=</span> <span class="cm-variable">x</span> <span class="cm-operator">+</span> <span class="cm-number">2</span>, <span class="cm-variable">f</span>(<span class="cm-variable">x</span>) <span class="cm-operator">=</span> <span class="cm-variable">x</span> <span class="cm-operator">*</span> <span class="cm-number">5</span>, <span class="cm-number">4</span>, <span class="cm-variable">f</span>(<span class="cm-variable">x</span>) <span class="cm-operator">=</span> <span class="cm-variable">x</span> <span class="cm-operator">-</span> <span class="cm-number">1</span> ] <span class="cm-operator">=&gt;</span> [ <span class="cm-number">25</span>, <span class="cm-number">3</span> ]</code></pre><p>because</p><pre><code class="language-javascript">[ (<span class="cm-number">3</span> <span class="cm-operator">+</span> <span class="cm-number">2</span>) <span class="cm-operator">*</span> <span class="cm-number">5</span>, <span class="cm-number">4</span> <span class="cm-operator">-</span> <span class="cm-number">1</span> ]</code></pre><p>Note how all functions collapsed into the closest number to the left; they did not produce new numbers into the array, but modified existing ones.</p><h1 id="notes">Notes</h1><ul><li>If a function comes first in the array, it should behave as if it were passed in <code>0</code>:<br><code>[ f(x) = x + 2, 4 ] =&gt; [0 + 2, 4]</code></li><li>An empty array passed in must return an empty array.</li><li>Functions will always take exactly one, number, parameter.</li><li>If an array only contains numbers, the array should be returned without any modifications.</li></ul>
