<h2><a href=https://www.codewars.com/kata/55df2e52954f1b311a000024/train/javascript target="_blank">Two variable simultaneous equations</a></h2><h3>6 kyu</h3><p>Given 2 simultaneous equations with 2 variables, find the value of the two variables.</p><p>The simultaneous equations will be handed to the function as a 2-d array, the non-zero integer coefficients of the corresponding variables will be in the same order for both of the inner arrays.  for example:</p><p>x + 2y = 11</p><p>//becomes --&gt; [1,2,11]</p><p>3x + 5y = 29</p><p>//becomes --&gt; [3,5,29]</p><p>Your function must return a 1-d array containing the values of the two variables in the order in which they were given in the argument. Solutions of all equations are integers. For the example above:</p><p>//should return an array in the form [x,y]</p><p>--&gt; [3,4]</p>
