<h2><a href=https://www.codewars.com/kata/56fd862db0c52efd9e0018f9/train/javascript target="_blank">ACME factory needs your help</a></h2><h3>6 kyu</h3><p>The ACME factory produces a bunch of assorted items, but unfortunately the software that controls the assembly lines has been corrupted. It's up to you to write the control logic</p><p>The factory takes as input an array of assembly lines. Each assembly line can be either a line of items (array) or an outside source (function). If it is an outside source, calling the function without any parameters will return an array of items.</p><p>Therefore each assembly line holds a number of items that can range from no items, to an arbitrary number of them. Each item on the assembly line is an object like the example below:</p><pre><code class="language-javascript">{<span class="cm-variable">item</span>: <span class="cm-string">'example item'</span>, <span class="cm-variable">priority</span>: <span class="cm-number">999</span>}</code></pre><p>The factory's job is to organize (sort) the items from highest priority (priority = 1) to lowest priority (Eg. priority = 1000), package the items into containers of 10, and make sure that only the items get shipped out (Ie. Just the item as a string, no priority).</p><p>Here is an example below:</p><pre><code class="language-javascript"><span class="cm-variable">factoryInput</span> <span class="cm-operator">=</span> [  [{<span class="cm-property">item</span>: <span class="cm-string">'wrench'</span>, <span class="cm-property">priority</span>: <span class="cm-number">1</span>}, {<span class="cm-property">item</span>: <span class="cm-string">'socket set'</span>, <span class="cm-property">priority</span>: <span class="cm-number">10</span>}],  <span class="cm-keyword">function</span> () { <span class="cm-keyword">return</span> [{<span class="cm-property">item</span>: <span class="cm-string">'cake'</span>, <span class="cm-property">priority</span>: <span class="cm-number">8</span>}, {<span class="cm-property">item</span>: <span class="cm-string">'glue'</span>, <span class="cm-property">priority</span>: <span class="cm-number">5</span>}] },]<span class="cm-variable">factoryOutput</span> <span class="cm-operator">=</span> [[<span class="cm-string">'wrench'</span>, <span class="cm-string">'glue'</span>, <span class="cm-string">'cake'</span>, <span class="cm-string">'socket set'</span>]]</code></pre><p>Notice that the output is an array of arrays. This is because the factory may have multiple "containers" (Max 10 items per container) depending on how many items it needs to ship.</p>
