<h2><a href=https://www.codewars.com/kata/5c1b25bc85042749e9000043/train/python target="_blank">Music Fun #1 - Major Scale</a></h2><h3>6 kyu</h3><p>Your music theory class just gave you a homework where you have to find the keys for a bunch of melodies.  You decided to be lazy and create a key detection program to do the work for you.</p><p>Each melody in this assignment is supposed to be strictly derived from a major scale. There are twelve major scales each of which starts from a differnt one of the twelve musical notes. The twelve musical notes are, in ascending order:</p><p><code>C, C#, D, D#, E, F, F#, G, G#, A, A#, B</code></p><p>(For simplicity, we only use sharps( # ) and not flats( b ))</p><p>A major scale has seven unique notes and follows a specific intervalic pattern.For instance, the ascending C major scale with notes C,D,E,F,G,A and B follows this pattern:</p><pre><code>Notes       C - D - E - F - G - A - B - CDistance      2   2   1   2   2   2   1</code></pre><p>The number indicated below each set of two notes shows the distance of the two notes. Between C and D, you have distance of 2 because you skip C#, but between E and F the distance is only 1.</p><p>The same pattern (2-2-1-2-2-2-1) occurs in every major scale.E major scale, which starts on the note E, goes:</p><pre><code>Notes       E - F# - G# - A - B - C# - D# - EDistance      2    2    1   2   2    2    1</code></pre><p>Your task is to create a function that takes a melody as a string and if the melody uses all the seven notes of a major scale, but no other notes, returns a string:</p><p><code>"? major scale"</code></p><p>Where <code>?</code> is the starting note of the scale (or the "key center").For instance, the input melody <code>"C#D#FF#D#FF#G#A#C"</code> returns <code>"C# major scale"</code>.</p><p>Beware! A melody doesn't always start on the first note of a scale. For example, <code>"D#BC#ABG#AF#E"</code> should return <code>"E major scale"</code>.</p><p>If the melody is not derived from any of the major scales, return:</p><p><code>"No major scale"</code></p><p>Note:</p><pre>- Input contains only -&gt; 'C','D','E','F','G','A','B', or '#' (no lower case)- No other letters, space, number or special characters are included- Even if a melody misses only one note from a scale, return "No major scale"(for instance, "CDEFGA" is not a complete C major scale)- Note names E# and B# are invalid, and input with these should return "No major scale"- Also invalid to have two sharps in a row (##), if this happens return "No major scale"</pre><ul><li>Also invalid to have two sharps in a row (##), if this happens, return "No major scale"</li><li>Input string that starts with a sharp should also return "No major scale"</li></ul>
