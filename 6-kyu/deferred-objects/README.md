<h2><a href=https://www.codewars.com/kata/53da21d54a51689580000e42/train/javascript target="_blank">Deferred Objects</a></h2><h3>6 kyu</h3><p>One of the most beautiful things about JavaScript is that it is asynchronous. And usually callbacks just work fine, but deferred objects allow to write asynchronous JavaScript parallel to how we write synchronous code and makes it more readable.</p><p>A deferred is an object representing work that is not yet done and a promise is an object representing a value that is not yet known.</p><p>Your task to create a Deferred class which should have the following properties</p><ul>    <li>promise :- the promise object. More details below</li>    <li>resolve(data) :- calls all the success handlers with the data</li>    <li>reject(data) :- calls all the failure handlers with the data </li></ul><p>Notice that once the deferred is complete ( resolve/reject is called once) then calling resolve or reject again should give an error</p><p>The promise object has three functions :- </p><ul>    <li>then(successHandler,errorHandler) :- Adds both the success and error Handlers. Error handler is optional</li>    <li>success(successHandler) :- Adds a success handler    </li><li>failure(errorHandler):- Adds a error handler</li></ul><p>Following is an example how your class will be used :-<br></p><pre><code class="language-javascript">  <span class="cm-keyword">var</span> <span class="cm-def">deferred</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">Deferred</span>();  <span class="cm-keyword">var</span> <span class="cm-def">promise</span> <span class="cm-operator">=</span> <span class="cm-variable">deferred</span>.<span class="cm-property">promise</span>;  <span class="cm-variable">promise</span>.<span class="cm-property">success</span>(<span class="cm-keyword">function</span>() {    <span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-string">"Success Handler Called"</span>);  });  <span class="cm-variable">promise</span>.<span class="cm-property">failure</span>(<span class="cm-keyword">function</span>() {   <span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-string">"Failure Handler Called"</span>);  });    <span class="cm-variable">deferred</span>.<span class="cm-property">resolve</span>(<span class="cm-string">"Success"</span>); <span class="cm-comment">// this will console.log "Success Handler Called"</span></code></pre>
