<h2><a href=https://www.codewars.com/kata/56d877bd97c0e584b2000053/train/javascript target="_blank">5x5 Tap Code</a></h2><h3>6 kyu</h3><h1 id="5x5-tap-code">5x5 Tap Code</h1><h2 id="introduction">Introduction</h2><p>I learned about this after watching 'The Flash' season 2 episode 13, there are 2 prisoners in adjacent cells communicating using what they call a 5x5 tap code. </p><p>The way it works is the user taps up to 5 times to signal a coresponding row and then once again to signal a corresponding column.</p><h3 id="example">Example</h3><p>Using a hypothetical 2x2 tap code the grid would be:</p><table><thead><tr><th>A</th><th>B</th></tr></thead><tbody><tr><td>C</td><td>D</td></tr><tr><td>Tapping <em><strong>twice</strong></em> then <em><strong>once</strong></em> would point to the letter 'C'.</td><td></td></tr><tr><td>Tapping <em><strong>once</strong></em> then <em><strong>once</strong></em> would point to 'A'...</td><td></td></tr></tbody></table><h2 id="instructions">Instructions</h2><p>Your task is to write a <code>tapFive</code> object that has two functions, <code>encrypt</code> and <code>decrypt</code> which uses a 5x5 tap code to decrypt and encrypt the passed in string. </p><h2 id="encrypt">Encrypt</h2><p>The <code>encrypt</code> method should take a string representing the word/sentence to encrypt, and return a string with the encrypted value. It can take a sentence with spaces (will ignore them) and the case shouldn't matter.</p><h3 id="example-1">Example</h3><pre><code class="language-javascript"><span class="cm-variable">tapfive</span>.<span class="cm-property">encrypt</span>(<span class="cm-string">"hello"</span>) <span class="cm-operator">===</span> <span class="cm-string">"2315313134"</span></code></pre><p>##Decrypt ##</p><p>The <code>decrypt</code> method should take a string representing an encoded word/sentence. It will contain only numbers such as <code>'2315313134'</code> for the word <code>'HELLO'</code>. The output will always be in caps.</p><h3 id="example-2">Example</h3><pre><code class="language-javascript"><span class="cm-variable">tapfive</span>.<span class="cm-property">decrypt</span>(<span class="cm-string">"2315313134"</span>) <span class="cm-operator">===</span> <span class="cm-string">"HELLO"</span></code></pre><h2 id="catch">Catch</h2><p>If you've noticed there are 26 letters in the alphabet (A...Z) and all must be encryptable, however in a 5x5 grid theres only enough room for 25 letters.</p><h3 id="hint--spoiler">Hint / Spoiler</h3><pre><code class="language-javascript"><span class="cm-variable">tapFive</span>.<span class="cm-property">encrypt</span>(<span class="cm-string">"JAWBREAKERS"</span>) <span class="cm-operator">===</span> <span class="cm-string">'2511521242151113154243'</span>;<span class="cm-variable">tapFive</span>.<span class="cm-property">decrypt</span>(<span class="cm-string">"2511521242151113154243"</span>) <span class="cm-operator">===</span> <span class="cm-string">"JAWBREACERS"</span>;</code></pre>
