<h2><a href=https://www.codewars.com/kata/562518e2059c742d6900007d/train/javascript target="_blank">Sparse array split function</a></h2><h3>6 kyu</h3><p>Sparse arrays in JavaScript are useful for many things. Among others they can be used for splitting an array into subarrays. Your task here is to create an array method for splitting a sparse array to subarrays, each of which must not be a sparse array. </p><p>It will act a bit similarly to the <code>.split()</code> method in strings. The new array method will return an array of subarrays, if the original array is a sparse array. Otherwise it will return the original array.<br>For example: <code>[1,2,,3,4].split()</code> will return <code>[[1,2],[3,4]]</code>, and <code>[,,,,].split()</code> will return <code>[]</code>.<br>Please note that nested sparse arrays should remain sparse: <code>[,,[,,],[,,],,].split()</code> should return <code>[[[,,],[,,]]]</code>.</p><p>Related kata:<br><a href="http://www.codewars.com/kata/sparse-array-this-is/" data-turbolinks="false" target="_blank">Sparse array this is?</a><br><a href="http://www.codewars.com/kata/sparse-array-implode-function/" data-turbolinks="false" target="_blank">Sparse array implode function</a></p>
