If you feel difficult, please complete the <a href="http://www.codewars.com/kata/56fe9b65cc08cafbc5000de3" data-turbolinks="false" target="_blank">simple version</a></p><h3 id="task">Task:</h3><p>  Give you three numbers:<code>a b c</code>,please return an equation(Operators include +,-,*,/), return type is string.</p><p>  For example: </p><p>  <code>sc(1,2,3)</code> should return <code>"1+2=3"</code> or <code>"2+1=3"</code>  or  <code>"3-2=1"</code>  or <code>"3-1=2"</code></p><p>  <code>sc(2,2,4)</code> should return <code>"2+2=4"</code> or <code>"2*2=4"</code>  or <code>"4/2=2"</code>  or <code>"4-2=2"</code></p><p>  <code>sc(6,2,3)</code> should return <code>"2*3=6"</code> or <code>"3*2=6"</code>  or <code>"6/2=3"</code> or <code>"6/3=2"</code></p><p>  Return one of the possible answer, you can pass the test.</p><p>  If there is <code>no equation between a,b,c</code>, return <code>""</code></p><p>You code length limit is 120 chars.</p><h3 id="code-length-calculation">Code length calculation</h3><p>  In javascript, we can't get the user's real code, we can only get the system compiled code. Code length calculation is based the compiled code.</p><p>  For example: </p><p>  If you typed <code>sc=x=&gt;x+1</code><br>  after compile, it will be:<code>sc=function(x){return x+1;}</code>
