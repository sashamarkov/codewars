<h2><a href=https://www.codewars.com/kata/55988922d24a02ccd0000063/train/python target="_blank">Write your own multiplication function</a></h2><h3>6 kyu</h3><p>This kata builds on <a href="http://www.codewars.com/kata/number-climber" data-turbolinks="false" target="_blank">Number Climber</a>.</p><p>For every positive number N there exists a unique sequence starting at 1 and ending at N, where each number in the sequence is either the double of the previous number or the double plus one. For example, if N = 13, then the sequence is [1, 3, 6, 13]. Given a number F, we can map the sequence by multiplying each number by F. So, for the example above and F = 7, we get the sequence <code>[1 * 7, 3 * 7, 6 * 7, 13 * 7] = [7, 21, 42, 91]</code>. Notice:</p><ol><li>The last number in the sequence is <code>N * F</code></li><li>Each number in the sequence is either the double of the previous number or the double plus F.</li></ol><p>Write a function <code>multiply</code> that takes two positive integers as arguments and returns the product of the two numbers. You are <em>only</em> allowed to use </p><ul><li>left and right shifts <code>&lt;&lt;, &gt;&gt;</code>, </li><li>addition and subtraction <code>+, -</code></li><li>logical operators <code>^, &amp;, |, ~</code> </li><li>comparisons <code>&lt;, &lt;=, &gt;=, &gt;</code>.</li></ul><p>You may <em>not</em> use the <code>*</code> operator or a builtin or imported multiplication function. </p><p>Also, you may not use any integer literal. Use the preprovided <code>zero</code> and <code>one</code> to denote the numbers 0 and 1 respectively. </p>
