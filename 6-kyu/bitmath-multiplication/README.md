<h2><a href=https://www.codewars.com/kata/57b68a8f20b1042eb900000e/train/javascript target="_blank">BitMath: Multiplication</a></h2><h3>6 kyu</h3><p>Create a utility function called <code>multiply</code> for your <code>BitMath</code> class, which will use bitwise operators (<strong>&amp;</strong>, <strong>|</strong>, <strong>^</strong>, <strong>~</strong>, <strong>&lt;&lt;**, **&gt;&gt;</strong>, and <strong>&gt;&gt;&gt;</strong>) to multiply 2 integers <code>a</code> and <code>b</code> without the use of the basic arithmetic operators (<strong>+</strong>, <strong>-</strong>, *****, <strong>/</strong>, and <strong>%</strong>).</p><p>If you've not yet trained on <a href="https://www.codewars.com/kata/bitmath-addition" data-turbolinks="false" target="_blank">BitMath: Addition</a>, you may want to start there.</p><p>As bitwise operation converts its operands to 32-bit signed integers, parameters in the test cases will be constrained such that:<br>-√(2<sup>31</sup> - 1) ≤ <code>a</code> ≤ √(2<sup>31</sup> - 1),<br>-√(2<sup>31</sup> - 1) ≤ <code>b</code> ≤ √(2<sup>31</sup> - 1)</p><p>Examples:<br><code>BitMath.multiply(2, 1)</code> --&gt; <code>2</code><br><code>BitMath.multiply(2, 2)</code> --&gt; <code>4</code><br><code>BitMath.multiply(6, 7)</code> --&gt; <code>42</code></p><p>Read up on bitwise operators here:<br><a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Bitwise_Operators" data-turbolinks="false" target="_blank">https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Bitwise_Operators</a></p><p>Note:<br>Don't comment your code, as <code>//</code> will be detected as a false positive instance of the division operator.The increment, decrement, unary negation, and unary plus operators are also disallowed, as well as the built-in JavaScript Math object.</p>
