<h2><a href=https://www.codewars.com/kata/57a4a5ca72292daeea0001c3/train/javascript target="_blank">T.T.T.21: Handle the text alignment</a></h2><h3>6 kyu</h3><h2 id="task">Task</h2><p>You are a teacher and here is a list of test scores. Please output them in accordance with the rules.</p><p>Complete function <code>formatScoreList()</code> that accept an object array <code>list</code>(contains the name and score of students). Output a string according to the format in the following example:</p><pre><code>list=[{name:"Tom",score:100},{name:"Jerry",score:95},{name:"Mike",score:90}]Output should be:+-----------------+--------+|      Name       | Score  |+-----------------+--------+|Tom              |100  pts|+-----------------+--------+|Jerry            |95   pts|+-----------------+--------+|Mike             |90   pts|+-----------------+--------+|The average score|95.0 pts|+-----------------+--------+</code></pre><h2 id="rules">Rules</h2><ol><li><p>The format of first line is: Centering alignment. If the number of spaces is not even, the right is a bit more than the left(1 space).</p></li><li><p>In other lines, the format of the left part (names list of students) is: Flush left. The width of this part depends on the longest string.</p></li><li><p>The format of the right part (scores list of students) is: Both ends aligned. The width of this part depends on the longest string. The average score should round to 1 decimal places.</p></li><li><p>In order to make Kata simple, you don't need to sort them ;-)</p></li></ol><h2 id="examples">Examples</h2><pre><code>list=[{name:"Tom",score:100},{name:"Jerry",score:95},{name:"Mike",score:90},{name:"Averyveryveryveryveryveryverylongname",score:1}]Output should be:+-------------------------------------+--------+|                Name                 | Score  |+-------------------------------------+--------+|Tom                                  |100  pts|+-------------------------------------+--------+|Jerry                                |95   pts|+-------------------------------------+--------+|Mike                                 |90   pts|+-------------------------------------+--------+|Averyveryveryveryveryveryverylongname|1    pts|+-------------------------------------+--------+|The average score                    |71.5 pts|+-------------------------------------+--------+list=[{name:"Tom",score:1},{name:"Jerry",score:2},{name:"Mike",score:3},{name:"Peter",score:4}]Output should be:+-----------------+-------+|      Name       | Score |+-----------------+-------+|Tom              |1   pts|+-----------------+-------+|Jerry            |2   pts|+-----------------+-------+|Mike             |3   pts|+-----------------+-------+|Peter            |4   pts|+-----------------+-------+|The average score|2.5 pts|+-----------------+-------+</code></pre>
