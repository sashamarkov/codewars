<h2><a href=https://www.codewars.com/kata/551115c5ce486ce707000487/train/javascript target="_blank">Hamming(7,4) code - Message encoding</a></h2><h3>6 kyu</h3><p>In telecomunications we use information coding to prevent errors while sending data. Depending on which type of coding is used we are able to detect erors on one or more bits of information and even to correct them. For more information check <a href="http://en.wikipedia.org/wiki/Hamming_code" data-turbolinks="false" target="_blank">http://en.wikipedia.org/wiki/Hamming_code</a>.</p><p>Hamming(7,4) is simple encoding method and it enable us to detect error in 1 bit and correct it, or to detect error in 2 bits (with not enough information to correct it).Encoding (and decoding) is explained here <a href="http://en.wikipedia.org/wiki/Hamming(7,4)" data-turbolinks="false" target="_blank">http://en.wikipedia.org/wiki/Hamming(7,4)</a></p><p>Your job is to write function encodeMessage which will encode message using Hamming (7,4) and lower chance for information corruption. </p><p>Input will always be string with length 4 containing only 0 and 1.</p><p>Example(<font color="red">Red are parity bits</font><font color="green"> green are data bits:</font>)<br>Message 1000 Encoded message: <font color="red">11</font><font color="green">1</font><font color="red">0</font><font color="green">000</font><br>Message 0001 Encoded message: <font color="red">11</font><font color="green">0</font><font color="red">1</font><font color="green">001</font></p>
