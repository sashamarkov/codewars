Where <code>start</code> is the start date of the interval and <code>end</code> is the end date of the interval.</p><p>For example:</p><pre><code class="language-javascript">  <span class="cm-keyword">var</span> <span class="cm-def">interval</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">TimeInterval</span>(<span class="cm-keyword">new</span> <span class="cm-variable">Date</span>(<span class="cm-string">'2012/03/24 14:00'</span>), <span class="cm-keyword">new</span> <span class="cm-variable">Date</span>());  </code></pre><p>The arguments of <code>new TimeInterval()</code> may be out-of-order:</p><pre><code class="language-javascript">  <span class="cm-keyword">var</span> <span class="cm-def">interval</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">TimeInterval</span>(<span class="cm-keyword">new</span> <span class="cm-variable">Date</span>(), <span class="cm-keyword">new</span> <span class="cm-variable">Date</span>(<span class="cm-string">'2012/03/24 14:00'</span>));  </code></pre><p>The objective of this <em>kata</em> is to implement the <code>intersection(interval)</code> and the <code>union(interval)</code> methods.</p><p><code>intersection(interval)</code> method receives a <em>TimeInterval</em> and returns a <em>TimeInterval</em> containing the matching time in both intervals.
