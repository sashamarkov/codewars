When invoked, the corosleeps for 1 sec and then returns a singlecharacter. </p><p>Your overall task is to implement an async<code>request_manager</code> which completes all <code>n</code> requestsin under <strong>1.5</strong> sec</p><hr><h2 id="summary">Summary:</h2><ul><li><p>Implement a coro <code>request_manager</code> which sends and processes fake requests</p><ul><li>It accepts the parameter <code>n</code> which represents the amount of requests that should be made</li></ul></li><li><p>Requests have to be made with <code>send_request</code></p><ul><li><p><code>send_request</code> is a preloaded coroutine : <code>(coro) send_request() -&gt; str</code></p></li><li><p>It takes 1 sec until a character is returned to the caller</p><ul><li>The waiting time is implemented via <code>asyncio.sleep</code></li></ul></li></ul></li><li><p>Characters returned by <code>send_request</code> have to be concat to one string</p><ul><li><p>That string is then returned by your <code>request_manager</code></p></li><li><p>The characters in the string need to be ordered, so that the character returned by the first request is at index 0, the character returned by the second request is at index 1 etc.</p></li></ul></li><li><p>Your <code>request_manager</code> has 1.5 sec to start &amp; process all <code>n</code> requests and to return the desired string</p></li></ul><hr><p>Good Luck, should be easy :)</p>
