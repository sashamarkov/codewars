<h2><a href=https://www.codewars.com/kata/55da2a419f8361df45000025/train/python target="_blank">Most improved - Puzzles #4</a></h2><h3>6 kyu</h3><h1 id="most-improved---puzzles-4">Most Improved - Puzzles #4</h1><p>When being graded in a subject or a course, high marks are focused on the most, but what about most improved? As a computer science teacher, you would like to create a function that calculates the most improved students and ranks them in a list.</p><h2 id="task">Task</h2><p>Your task is to complete the function to return an array sorted by most improved as percentages.</p><h2 id="input">Input</h2><p>The input you will receive will be an array of students, students will be objects containing a name and an array of marks (in order of acheived). The marks will be out of 100, a student can however have a mark of null if the test was not attempted (treat this as 0).</p><p><strong>Example of student Object:</strong> <code>{name:'Henry, Johns',marks:[25,50]}</code></p><h2 id="output">Output</h2><p>The output expected will be an array of objects similar to the student object, containing the name and total improvement percentage out of the first and last mark given to calculate the overall improvement percentage. The output array must be sorted by most improved (Round the calculated improvement). If there is a tie in improvements, then order by name (capitals before lowercase).</p><p><strong>Example of return Object:</strong> <code>{name:'Henry, Johns',improvement:100}</code></p>
