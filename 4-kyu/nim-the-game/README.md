<h2><a href=https://www.codewars.com/kata/54120de842dff35232000195/train/python target="_blank">NIM the game</a></h2><h3>4 kyu</h3><p>This kata explores writing an AI for a two player, turn based game called <em>NIM</em>.</p><h2 id="the-board">The Board</h2><p>The board starts out with several piles of straw.  Each pile has a random number of straws.</p><pre><code>Pile 0: ||||Pile 1: ||Pile 2: |||||Pile 3: |Pile 4: ||||||...or more concisely: [4,2,5,1,6]</code></pre><h2 id="the-rules">The Rules</h2><ul><li>The players take turns picking a pile, and removing any number of straws from the pile they pick</li><li>A player must pick at least one straw</li><li>If a player picks the last straw, she wins!</li></ul><h2 id="the-task">The Task</h2><p>In this kata, you have to write an AI to play the straw picking game.</p><p>You have to encode an AI in a function <code>choose_move</code> (or <code>chooseMove</code>, or <code>choose-move</code>) that takes a board, represented as a list of positive integers, and returns</p><pre><code class="language-python">(<span class="cm-variable">pile_index</span>, <span class="cm-variable">number_of_straws</span>)</code></pre><pre style="display: none;"><code class="language-haskell">(<span class="cm-variable">pileIndex</span>, <span class="cm-variable">numberOfStraws</span>)</code></pre><pre style="display: none;"><code class="language-javascript">[<span class="cm-variable">pile_index</span>, <span class="cm-variable">number_of_straws</span>]</code></pre><pre style="display: none;"><code class="language-clojure"><span class="cm-bracket">[</span><span class="cm-variable">pile-index</span>, <span class="cm-variable">number-of-straws</span><span class="cm-bracket">]</span></code></pre><p>Which refers to an index of a pile on the board, and some none-zero number of straws to draw from that pile.</p><p>The test suite is written so that your AI is expected to play 50 games and win every game it plays.</p>
